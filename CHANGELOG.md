# Changelog

All notable changes to DeepCaller will be documented in this file.

---

## [1.0.0] — 2026-05-15

### Initial release

- Deep learning-based variant calling for tetraploid and hexaploid genomes
- Five pre-trained models: potato, alfalfa, rose, sweetpotato, syn_potato
- Speed and performance inference modes per species
- Whole-genome depth normalisation for robust variant filtering
- BED file support for targeted variant calling
- bgzip-compressed, tabix-indexed VCF 4.2 output
- GPU acceleration support (optional)
